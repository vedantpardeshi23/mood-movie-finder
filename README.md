MOOD BASED MOVIE FINDER :-

📌 Project Overview - 

This project is a web application that recommends movies based on user's mood. Instead of searching manually, user can select their current mood and get relevant movie suggestions to choose from.

The idea behind this project is to make boring movie selections more interactive and personalized by connecting moods with movie genres.

🎯 Features :-

1. Select a particular mood (Happy , sad , excited , etc).
2. Fetch movie data from an API (From Tmdb).
3. Display movie posters, titles, and ratings.
4. Search movies by name.
5. Filter movies by genre.
6. Sort movies based on rating or popularity.
7. Responsive design for different screen sizes.

🌐 API USED :- 

The project uses "The Movie Database (Tmdb)" API to fetch the data of the movies.

📝 Methods / Files used :-

1. index.html (For html)
2. style.css (for designing)
3. script.js (for logic) :-
    a. Fetch API
    b. Array Higher Order Functions 

💡 Project Goal

The goal of this project is to understand how to:

1. Work with APIs using fetch
2. Handle and display dynamic data
3. Use array methods for searching, filtering, and sorting
4. Build a responsive and interactive UI
